# Swara Maternity Clinic - Premium Booking System 🌸

An elegant, secure, end-to-end full-stack appointment booking application custom-built for **Dr. B Pruthvi**. This system features a soothing, boutique-style user experience tailored specifically for expectant mothers, paired with a frictionless, zero-login email-driven administration workflow for the doctor.

---

## 🏗️ Architecture & Project Structure

This repository is structured as a unified monorepo:

* **backend/** - Node.js & Express server, automated mail engine
* **frontend/** - Vite + React interface with Tailwind CSS styling

---

## ✨ Key Features

### 🌸 For the Patient (The Frontend UI)
* **Boutique Visual Identity:** Soft rose-gold blush, cream whites, and warm stones to cultivate an instantly calming, high-end care environment.
* **Biographical Storytelling:** Dedicated sections for Dr. Pruthvi's medical milestones, credentials, and care philosophies.
* **Smart Appointment Requests:** Form utilizing localized operational cards, integrated calendar mapping slots, and active auto-fill browser safety overrides.

### 🚨 For the Doctor (The Approval Engine)
* **Inbox Dashboard Workflow:** The doctor accepts or declines real-time booking requests instantly from their phone's email inbox using secure action links—no complex administrative panel logins required.
* **Double-Email Automation:** Generates immediate action alerts straight to the doctor, and automatically routes customized verification cards to the patient upon approval.

---

## 🛠️ Tech Stack & Dependencies

* **Frontend:** React, Vite, Tailwind CSS, Lucide React (Icons)
* **Backend:** Node.js, Express.js, Nodemailer (SMTP Routing)
* **Database:** MongoDB (Structured Collection Schemas)
